## How to Start Backend
1. `cd backend`
2. `venv\Scripts\activate` (Windows) or `source venv/bin/activate` (Mac)
3. `pip install -r requirements.txt`
4. `python manage.py runserver`

## How to Start Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
