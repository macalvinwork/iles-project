# Internship Logging and Evaluation System (ILES)
## User Stories: Academic Supervisor & Internship Administrator

### Academic Supervisor
- **A-01:** Log in and see assigned students – Only assigned students visible.
- **A-02:** Score students using defined criteria – All fields required before submission.
- **A-03:** Auto-compute final scores – Backend computes total score automatically.

### Internship Administrator
- **I-01:** Create user accounts and assign roles – Role cannot be changed after creation.
- **I-02:** Assign placements – All fields required; no overlapping dates.
- **I-03:** Define evaluation criteria and weights – Total weights must equal 100%.
- **I-04:** Dashboard showing system progress – Displays key metrics.
- **I-05:** Generate reports – Reports are filterable and exportable.
