# Internship Logging and Evaluation System (ILES)
## User Stories: Student Intern & Workplace Supervisor

### Student Intern
- **S-01:** Register and log in – Login fails with wrong password. Different dashboards per role.
- **S-02:** See placement details (company, supervisor, dates) – Visible only after Admin assigns it.
- **S-03:** Write weekly log and save draft – Draft logs not visible to supervisor; editable freely.
- **S-04:** Submit log before deadline – Submission blocked after deadline; status changes from Draft to Submitted.
- **S-05:** See feedback on returned log – Returns to Draft with supervisor comments; editable for resubmission.
- **S-06:** View final evaluation score – Scores visible only after evaluation.

### Workplace Supervisor
- **W-01:** Log in and see assigned interns – Can only see assigned interns.
- **W-02:** View submitted logs – Only Submitted logs visible.
- **W-03:** Approve log with comment – Status changes to Reviewed; comment saved and visible.
- **W-04:** Return log for revision with comment – Comment required; status returns to Draft.
- **W-05:** View log history – Shows status, user, timestamp, and comments.
